# 🔨 OpenClaw Starter Kit V2

Welcome! You're about to set up your own AI assistant — one that lives on your computer, talks to you through Telegram, and can search the web, manage files, run code, and a whole lot more.

This kit walks you through every step, in plain English, no prior experience required. If you've never opened a terminal before, that's completely fine. We'll get there together.

---

## What Is OpenClaw?

OpenClaw is an open platform for running AI assistants on your own machine. Instead of using a generic chatbot on a website, you get:

- 🤖 **Your own assistant** — with a name, personality, and memory you define
- 💬 **Chat from anywhere** — via Telegram on your phone or the web dashboard
- 🔒 **Your data stays yours** — everything runs on your computer, not someone else's server
- 🛠️ **Expandable** — add skills for weather, reminders, code, and more

---

## 📋 Pre-Flight Checklist

Before you start, make sure you have:

- [ ] A **Windows PC** (Windows 10 version 2004 or later, or Windows 11)
- [ ] An **internet connection**
- [ ] About **45–60 minutes** of uninterrupted time for the first setup
- [ ] A **notepad** (physical or digital) to save passwords and keys as you create them
- [ ] Your **phone** nearby (you'll need it to set up Telegram)
- [ ] A **credit or debit card** (for the Anthropic AI account — you only pay for what you use, typically $5-20/month)

> 💡 **Best way to read these guides:** View them on GitHub at the link you were given — the formatting will be clean and all links will work. If you downloaded the files instead, you can open `.md` files in any text editor (Notepad works), but they won't look as pretty.
